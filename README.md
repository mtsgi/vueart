# Vue Art Designer

懐かしい画像編集ソフトのUIと操作感をリスペクトした、Webベースのベクター画像編集ツールです。

## 概要

Vue 3 + TypeScript + Pinia で実装した MDI（マルチドキュメントインタフェース）型のSVGベクター画像エディタです。

## 主な機能

- **MDIワークスペース** — 複数のキャンバスウィンドウを同時に開いて編集
- **ベクター描画** — 矩形・楕円・テキストオブジェクトの配置・編集
- **オブジェクトスタック** — レイヤー順の管理と表示/ロック切り替え
- **SVGフィルタ** — ベベル・ドロップシャドウ・内側グロー・エンボスなどの視覚効果
- **アンドゥ/リドゥ** — Ctrl+Z / Ctrl+Y によるキーボード操作
- **エクスポート** — PNG・SVG形式でのダウンロード
- **素材サイドバー** — 基本図形・SVGフィルタ素材をフォルダツリーで管理
- **[イメージ]ツールバー** — 選択オブジェクトの色・線幅・不透明度をインライン編集

## UIレイアウト

## 技術スタック

| 項目 | 詳細 |
|------|------|
| フレームワーク | Vue 3.5.x（Composition API / `<script setup>`） |
| 言語 | TypeScript ~6.0（strict + erasableSyntaxOnly） |
| 状態管理 | Pinia |
| スタイル | SCSS |
| ビルドツール | Vite 8.x |
| レンダリング | SVG（Canvas APIは不使用） |

## 開発環境のセットアップ

```bash
npm install
npm run dev      # 開発サーバー起動 (http://localhost:5173)
npm run build    # プロダクションビルド
npm run preview  # ビルド結果のプレビュー
```
